Fictional computation method

This prototype uses a deliberately simplified present-dollar model. It is not a real engineering estimate.

  • Each collector represents a fictional industrial package: mining, refining, launch, orbital placement, and control systems.
  • Base cost starts at roughly $120 billion per collector-equivalent package, then falls as automation and launch infrastructure improve.
  • Time advances faster than real time so centuries of construction can be shown in minutes.
  • Power captured scales with collector count and solar proximity, but remains symbolic.
  • Venus degradation is visualized as a material-use percentage, not literal planetary physics.

The key idea respected here: this is a swarm of independent orbiting collectors, not a solid shell.
